Ruta Básico

Empieza con lo esencial: estructura de un buen prompt, objetivos claros, formato de salida y control de contexto.

Conceptos clave

  • Define rol, objetivo y restricciones.
  • Pide formato de salida claro: Markdown, JSON o pasos.
  • Evalúa tu prompt: ¿qué falta?, ¿qué sobra?, ¿qué ambigüedades hay?

Ejercicio guiado rápido

  1. Abre Plantillas y descarga instrucciones_base.md.
  2. Rellena los huecos entre corchetes.
  3. Pruébalo en tu modelo favorito.

La API real la añadiremos más adelante. Por ahora, céntrate en la calidad del prompt.

Checklist de calidad de prompt

  • Propósito definido en una frase.
  • Formato de salida y límites de longitud.
  • Datos de ejemplo y contraejemplos.
  • Evaluación: criterios de “listo para producción”.
