# Reglas Globales

Este archivo contiene las instrucciones de alto nivel que el agente debe seguir en todo momento, independientemente del proyecto o la habilidad específica que esté utilizando.

## 🇪🇸 Idioma y Comunicación
- **Toda la documentación** generada para el usuario (leame, planes de implementación, walkthroughs, etc.) debe redactarse obligatoriamente en **español**.
- Los **comentarios en el código** y la documentación técnica dentro del código deben ser en español para facilitar la comprensión del equipo.
- Las respuestas directas al usuario y las explicaciones de cambios deben mantener un tono profesional y cercano en español.

## 📁 Estructura y Estándares
- Seguir siempre la estructura oficial de habilidades de Antigravity (`SKILL.md`, `scripts/`, `resources/`).
- Priorizar la reutilización de habilidades existentes antes de crear nuevas desde cero.

## 📜 Memoria de Preferencias
- La preferencia de idioma es **Español** (Establecida el 2026-03-05).
