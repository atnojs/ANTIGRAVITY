# Style Guide Skill (Guía de Estilo)

Esta habilidad proporciona un conjunto completo de recursos de diseño y plantillas CSS para crear aplicaciones con un estilo **Neon Glassmorphism**.

## Recursos

Todos los recursos se encuentran en el directorio `resources/` de esta habilidad.

- **base.css**: Variables CSS, colores, gradientes, tipografía.
- **components.css**: Efectos de cristal (glassmorphism), botones 3D, tarjetas interactivas, inputs, alertas.
- **hover-effects.css**: Efectos de hover animados.
- **login-modal.css**: Estilos premium para modales de inicio de sesión.
- **loaders.css**: Spinners e indicadores de carga.
- **examples.css**: Snippets de código JSX/React.
- **index-templates.html**: Plantilla HTML con dependencias incluidas.

## Uso

Al construir una aplicación web o implementar una interfaz de usuario:

1.  **Referenciar los Recursos**: Lee los archivos en `resources/` para entender las clases y variables disponibles.
2.  **Aplicar Estilos Genéricos**: Comienza usando `base.css` para configurar las variables de color y la tipografía.
3.  **Usar Componentes**: Implementa elementos de UI usando las clases de `components.css`, `login-modal.css` y `loaders.css`.
4.  **Seguir el Diseño**: Asegúrate de mantener la estética "Neon Glassmorphism" (fondos oscuros, acentos brillantes, efectos de cristal desenfocado).
