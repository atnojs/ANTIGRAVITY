# Plantilla de Skill

Esta es la estructura oficial para trabajar con skills según la documentación de Google Antigravity.

## Estructura del Directorio
- **SKILL.md**: Este archivo. Es el "cerebro" de la skill, contiene este frontmatter (YAML) y las instrucciones en Markdown. Las skills se cargan bajo demanda.
- **scripts/**: Carpeta (opcional) para scripts en Python, Bash, Node.js u otros que la skill necesite ejecutar.
- **resources/**: Carpeta (opcional) para plantillas, snippets de documentación, o archivos de configuración de soporte.
