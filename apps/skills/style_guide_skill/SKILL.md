# Style Guide Skill

This skill provides a complete set of design resources and CSS templates for creating applications with a **Neon Glassmorphism** style.

## Resources

All resources are located in the `resources/` directory of this skill.

- **base.css**: CSS variables, colors, gradients, typography.
- **components.css**: Glass effects, 3D buttons, interactive cards, inputs, alerts.
- **hover-effects.css**: Animated hover effects.
- **login-modal.css**: Premium login modal styles.
- **loaders.css**: Spinners and loading indicators.
- **examples.css**: JSX/React code snippets.
- **index-templates.html**: HTML template with dependencies.

## Usage

When building a web application or implementing a UI:

1.  **Reference the Resources**: Read the files in `resources/` to understand the available classes and variables.
2.  **Apply Generic Styles**: Start by using `base.css` to set the color variables and typography.
3.  **Use Components**: Implement UI elements using the classes from `components.css`, `login-modal.css`, and `loaders.css`.
4.  **Follow the Design**: Ensure the "Neon Glassmorphism" aesthetic is maintained (dark backgrounds, glowing accents, blurred glass effects).
