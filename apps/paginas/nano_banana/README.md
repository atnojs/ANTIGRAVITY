## Instalación (Hostinger)
1) Sube este ZIP a `public_html/app-ia-v2/` y descomprímelo.
2) En hPanel → Advanced → **Environment**, añade `GOOGLE_API_KEY=TU_CLAVE` (o pega la clave en los PHP reemplazando `TU_API_KEY`).
3) Abre `https://tudominio/app-ia-v2/index.html`.

Si ves solo texto tipo “v2 UI placeholder”, significa que subiste un paquete incompleto. Reemplaza **todo** con este ZIP.
