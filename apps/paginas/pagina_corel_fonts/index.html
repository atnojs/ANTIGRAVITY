Guía Interactiva de Corel Font Manager

Una guía completa y moderna para dominar la gestión de tipografías. Navega por las secciones para descubrir todo su potencial.

Introducción

Corel Font Manager es una aplicación independiente de administración de fuentes que viene incluida en CorelDRAW Graphics Suite, diseñada para simplificar la localización, organización y gestión de tipografías. Permite un control total sobre el flujo de trabajo tipográfico, desde la instalación hasta la organización en colecciones personalizadas.

Inicio

Puedes lanzar Corel Font Manager de dos maneras:

  • Directamente como cualquier otra aplicación desde tu sistema operativo.
  • Desde CorelDRAW, accediendo al "Iniciador de aplicaciones" en la barra de herramientas estándar y seleccionando Corel Font Manager.

Interfaz de Usuario

La interfaz se organiza en varios paneles clave para una gestión eficiente:

Panel Bibliotecas

Situado a la izquierda, este panel muestra las bibliotecas de fuentes, incluyendo carpetas y colecciones que hayas añadido. Permite filtrar fuentes y cuenta con botones para añadir o eliminar carpetas y colecciones.

Panel de Previsualización

Aquí se visualizan las fuentes de la biblioteca seleccionada con texto predefinido o personalizado. Ofrece herramientas para instalar, desinstalar y gestionar fuentes.

Panel de Propiedades

Este panel detalla los conjuntos de glifos y la información de la fuente, incluyendo la posibilidad de ver y ajustar las propiedades de las fuentes variables.

Búsqueda y Previsualización

Navega y encuentra fuentes fácilmente con las siguientes herramientas:

  • Navega por tus fuentes seleccionando una biblioteca en el Panel Bibliotecas.
  • El Panel de Previsualización muestra las fuentes con indicadores visuales de estado (instalada, no instalada, protegida, etc.).
  • Previsualiza fuentes con texto genérico o escribe tu propio texto para una vista previa personalizada.
  • Utiliza palabras clave como nombre, tipo de tecnología (OpenType, TrueType), grosor o fabricante en el cuadro de lista de fuentes para una búsqueda rápida.

Instalación y Desinstalación

Gestiona las fuentes instaladas en tu sistema de forma sencilla.

Instalar Fuentes

  • Selecciona una o varias fuentes (con Ctrl o Shift) y haz clic en "Instalar".
  • Un indicador amarillo junto a la fuente cambiará a verde al instalarse.

Desinstalar Fuentes

  • El proceso es el mismo que para instalar. El indicador verde volverá a ser amarillo.

Restricciones

Las fuentes protegidas por el sistema operativo o aquellas en uso por otras aplicaciones no pueden ser desinstaladas hasta que se cierren dichas aplicaciones o se reinicie el equipo.

Eliminación de Fuentes

Corel Font Manager permite eliminar permanentemente las fuentes que ya no necesitas de tu sistema, ayudando a mantener tu biblioteca de fuentes limpia y ordenada. Esta acción es diferente a la desinstalación, ya que borra los archivos de fuente.

Carpetas y Colecciones

Organiza tus fuentes de manera lógica y eficiente.

Carpetas

Facilitan el acceso y monitoreo de fuentes en directorios específicos sin necesidad de instalarlas.

  • Añadir Carpeta: Haz clic en "Añadir carpeta" y selecciona un directorio.
  • Ventaja: Usa fuentes en CorelDRAW sin instalarlas, optimizando el rendimiento. Los cambios en la carpeta se reflejan al reiniciar el gestor.

Recomendación

Es preferible monitorear carpetas que contengan exclusivamente archivos de fuentes para evitar problemas de rendimiento.

Colecciones

Ideales para agrupar fuentes de distintas ubicaciones para un proyecto específico, sin duplicar archivos.

  • Crear Colección: Haz clic en "Añadir colección", asígnale un nombre y arrastra las fuentes deseadas.
  • Puedes renombrar y eliminar fuentes de las colecciones fácilmente.

Exportar e Importar

Puedes exportar e importar tus carpetas y colecciones como un archivo XML (Archivo > Exportar/Importar carpetas y colecciones) para compartirlas o hacer una copia de seguridad.

Acceso a Google Fonts

Los suscriptores de CorelDRAW Graphics Suite (2021.5 en adelante) tienen acceso directo a más de 1000 familias de fuentes de Google Fonts.

  • Explora, busca y previsualiza estas fuentes en línea sin instalarlas.
  • Al usar una fuente remota en CorelDRAW o PHOTO-PAINT, se descarga automáticamente.
  • Puedes optar por ocultar las fuentes remotas si prefieres trabajar solo con tu biblioteca local.

Fuentes Variables

Corel Font Manager es compatible con fuentes variables, que integran una familia tipográfica completa en un solo archivo. Estas fuentes se identifican con un icono distintivo y puedes visualizar y ajustar sus propiedades (como grosor y estilo) directamente en el panel de propiedades.

Gestión de Duplicados

Mantén tu biblioteca de fuentes sin redundancias.

  • Corel Font Manager identifica fuentes duplicadas con una insignia.
  • Por defecto, solo se muestra una versión de cada grupo de duplicados.
  • Puedes activar la visualización de todos los duplicados para eliminar las versiones innecesarias. Solo una versión puede estar instalada a la vez.

Visualización de Glifos

Explora todos los caracteres de una fuente. El panel de propiedades te permite ver los glifos de una fuente y aplicar filtros para subconjuntos específicos (comunes, script, funciones OpenType). Además, puedes consultar información detallada de la fuente, como peso, ancho, derechos de incrustación, fabricante y copyright.

Consejos Adicionales

Consideraciones importantes para optimizar tu flujo de trabajo y el rendimiento del sistema.

Rendimiento

Un número excesivo de fuentes instaladas puede afectar el rendimiento. Usa la función de monitoreo de carpetas para acceder a fuentes sin instalarlas.

Fuentes Type 1

Este formato (PostScript) es obsoleto. Se recomienda usar TrueType (TTF) u OpenType (OTF) para evitar problemas de compatibilidad.

Sustitución de Fuentes

La concordancia PANOSE ayuda a sustituir fuentes faltantes por similares. CorelDRAW también puede buscar y descargar fuentes faltantes en línea.

Incorporación de Fuentes

Al guardar un dibujo, puedes incorporar las fuentes utilizadas para que otros usuarios puedan ver e imprimir el documento sin tenerlas instaladas.

Personalización y Restablecimiento

Puedes personalizar la interfaz. Si tienes problemas, puedes restablecer la base de datos de fuentes, lo que la recrea desde cero (se eliminarán tus colecciones y carpetas añadidas).
