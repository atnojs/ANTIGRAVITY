2. El "Prompt Maestro" (Especificación del Proyecto)
Este es el resumen de la lógica que aplicaré a tus archivos:

Objetivo: Transformar una galería PHP local en una Web App moderna (SPA) integrada con Firebase y Gemini 3 Pro Image.

Funcionalidades Core:

Módulo de Imagen (Home): Campo de carga de archivos que sube la foto a Firebase Storage. Botón "Generar" conectado a la API de Google AI Studio (Gemini 3) que envía la imagen original + el prompt seleccionado.

Flujo de Galería Inteligente: Sistema de redirección que captura el prompt de la galería y lo inyecta en el textarea de la Home mediante parámetros de búsqueda en la URL.

Base de Datos (Firebase): Migración del archivo estado_galeria.json a Firestore para que las categorías y prompts sean editables en tiempo real sin tocar código.

Descarga de Resultado: El área de resultado mostrará la imagen generada por Gemini con un botón de descarga directa y opción de guardado en la cuenta del usuario.

Modo Admin: Reemplazo del sistema de contraseña simple por Firebase Authentication para gestionar quién puede subir o editar prompts en la galería.
